<!-- ### 1. **Universal Pattern Framework**
Every pattern is a grid where you decide what to print at each (row, col) position.

### 2. **Triangle Formulas**
- Increasing: `elements = i`
- Decreasing: `elements = n - i + 1`
- Right-aligned: `spaces = n - i`

### 3. **Pyramid Formulas**
- Stars per row: `2 * i - 1` (odd number sequence)
- Leading spaces: `n - i`

### 4. **Hollow Pattern Strategy**
Print symbol when on **boundaries**, print space otherwise.
```java
if (i == 1 || i == n || j == 1 || j == n) // Boundary
```

### 5. **Diagonal Detection**
- Main diagonal: `i == j`
- Anti-diagonal: `i + j == n + 1`

### 6. **Spiral Algorithm**
Track four boundaries (top, bottom, left, right) and shrink inward after each direction.

### 7. **Pattern Composition**
Complex patterns = Combining multiple simple conditions with OR (||) -->
